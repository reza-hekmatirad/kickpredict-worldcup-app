**KickPredict**



KickPredict is a full-stack World Cup match predictor application.



Users can browse teams, view matches, create score predictions, and manage prediction data through a custom REST API.

**Main Features**


- View World Cup teams
- View match schedule
- Create, edit and delete score predictions
- Filter teams and matches
- Responsive frontend
- REST API with documented endpoints





**Tech Stack**



\- Frontend: HTML, CSS, JavaScript

\- Backend: Node.js, Express

\- Database: SQLite
